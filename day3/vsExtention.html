What is VSCode?

Visual Studio Code, also commonly referred to as VS Code, is an integrated development environment developed by Microsoft for Windows, Linux, macOS, and web browsers. Visual Studio Code was first announced on April 29, 2015, by Microsoft at the 2015 Build conference. A preview build was released shortly thereafter.

On November 18, 2015, the project "Visual Studio Code — Open Source" (also known as "Code — OSS"), on which Visual Studio Code is based, was released under the open-source MIT License and made available on GitHub. Extension support was also announced. On April 14, 2016, Visual Studio Code graduated from the public preview stage and was released to the web.

Features

  • IntelliSense:

    • VSCode goes beyond the usual syntax highlighting and offers IntelliSense, i.e. smart completions tailored to the code and its intricacies.

  • Run and Debug:

    • VSCode allows you to run your code (with or without debugging it with the help of a web editor) in the app itself.

  • Built-in Git:

    • Working with Git is made easier in VSCode which allows you to make commits, stage files, and other Git actions right from the editor.

  • Extension:

    • Personalization of your coding environment has never been easier with over 30,000 extensions present to do anything from adding new languages, themes, etc., to contracting any additional service such as AI integration.

Extensions

Extensions in VSCode add extra functionality to the editor, improving the development experience by providing tools, code snippets, themes, linters, debuggers, and more. Extensions can be installed directly from the VSCode marketplace.

S.no Name of the Extension Features
1. Prettier - Code Formatter
  • Auto-formats code on save.
  • Supports various languages (JavaScript, TypeScript, JSON, CSS, etc.)
  • Configurable formatting rules.
2. GitLens
  • View Git blame annotations directly in the code.
  • Visualize Git commit history and changes.
  • Compare branches, commits, and file versions
  • Search through Git history.
3. GitHub Pull Requests and Issues
  • View, create, and manage pull requests.
  • Review and comment on pull requests directly from VSCode.
  • View and create issues.
4. Bracket Pair Colorizer
  • Different colors for each pair of matching brackets.
  • Supports curly braces {}, parentheses (), square brackets [], etc.
  • Easily configurable to adjust bracket colors.
6. Auto Rename Tag
  • Renames both the opening and closing tags at the same time.
  • Works in HTML, XML, JSX, and other tag-based languages.
  • Easy to use and saves time when editing markup.
7. CSS Peek
  • Peek the CSS rules associated with HTML elements.
  • Jump to the exact location of the CSS rule definition.
  • Works with both external and embedded CSS.

EMMETS

Emmet is a popular toolkit for web developers, primarily used to speed up the process of writing HTML and CSS. Emmet provides a shorthand syntax for quickly generating code snippets, and it's integrated into VSCode by default.

Use Emmet:

Type div.container>ul>li*3 and press Tab to generate the HTML structure:

S.no EMMETS Examples
1. Child (>) and Sibling (+) Operators div>ul>li*3
2. Multiplication (*) li*5
3. Attributes ([])
  • a[href="link"]
  • input[type="text"]
  • img[src="image.jpg" alt="image description"]
4. Text Content ({}) h1{Hello World}
5. Grouping and Nesting Elements div>(ul>li)*3
